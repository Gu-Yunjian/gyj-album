# GU-Album 设计规范

## 一、设计原则

### 1. 极简主义
- 大留白，减少视觉噪音
- 文字不显眼，图片为主
- 去除不必要的装饰元素

### 2. 沉浸感
- 照片是核心，内容优先
- 交互平滑，不打断浏览体验
- 弱化 UI 控件的存在感

### 3. 一致性
- 全局统一的视觉语言
- 统一的交互模式
- 统一的动画节奏

---

## 二、色彩规范

### 1. 亮色模式 (Light)
```css
--bg-primary: #FFFFFF;      /* 主背景 - 纯白 */
--bg-secondary: #FAFAFA;    /* 次级背景 */
--bg-card: #FFFFFF;         /* 卡片背景 */

--text-primary: #1A1A1A;    /* 主文字 - 近黑 */
--text-secondary: #666666; /* 次级文字 */
--text-muted: #999999;      /* 弱化文字 */

--border: #E5E5E5;          /* 边框 */
--shadow: rgba(0, 0, 0, 0.08); /* 阴影 */
```

### 2. 暗色模式 (Dark)
```css
--bg-primary: #0A0A0A;      /* 主背景 - 近黑 */
--bg-secondary: #141414;    /* 次级背景 */
--bg-card: #1A1A1A;         /* 卡片背景 */

--text-primary: #F5F5F5;    /* 主文字 - 近白 */
--text-secondary: #A0A0A0; /* 次级文字 */
--text-muted: #666666;      /* 弱化文字 */

--border: #2A2A2A;          /* 边框 */
--shadow: rgba(0, 0, 0, 0.4); /* 阴影 */
```

### 3. 强调色
```css
--accent: #333333;           /* 强调色 - 深灰 (不抢图片风头) */
--accent-hover: #000000;    /* hover 态 */
```

## 三、字体规范

### 1. 主字体
- **字体**: MiSans (开源思源黑体)
- **引入**: 思源黑体官方 CDN

### 2. 字重层级
| 用途 | 字重 | 大小 (PC) | 大小 (移动) |
|------|------|-----------|-------------|
| 姓名/标题 | 500 (Medium) | 32px | 24px |
| 副标题 | 400 (Regular) | 18px | 16px |
| 正文 | 400 (Regular) | 15px | 14px |
| 辅助文字 | 300 (Light) | 13px | 12px |
| 导航 | 400 (Regular) | 14px | 14px |

### 3. 中文排版
- `letter-spacing: 0.02em` (字间距)
- `line-height: 1.8` (行高)
- 中英文混排时注意间距

### 1. 基础单位
```css
--space-xs: 4px;

--space-md: 16px;
--space-lg: 24px;
```

- **PC**: 48px - 64px
- **移动**: 16px - 24px

## 五、组件规范

### 1. 照片卡片 (PhotoCard)
- 白色卡片包裹 (`bg-card`)
- 边框圆角: `border-radius: 4px`
- 内边距 (留白): 自适应，照片四周 16px-32px
- 阴影: 柔和投影
- **hover**: 轻微上浮 + 阴影加深

### 2. 画廊画布 (GalleryCanvas)
- 可无限拖动
- 拖动惯性: 缓动减速
- 容器背景: 透明或微弱纹理

### 3. 导航 (Navigation)
- 位置: 顶部固定
- 左侧: 站名 + 导航链接
- 右侧: 主题切换
- 风格: 极简文字链接
- hover: 微弱变色

### 4. 灯箱 (Lightbox)
- 全屏黑色遮罩 (rgba(0,0,0,0.95))
- 居中显示照片
- 关闭按钮: 右上角
- 切换: 左右箭头 + 键盘支持

### 5. 影集卡片 (CollectionCard)
- 封面照片
- hover: 右下角浮层显示信息
- 浮层背景: 半透明黑

---

## 六、动画规范

### 1. 过渡时长
| 类型 | 时长 | 缓动 |
|------|------|------|
| 基础 | 200ms | ease-out |
| 页面 | 300ms | ease-in-out |
| 拖动 | 惯性 | ease-out |

### 2. 动画类型
- **淡入**: opacity 0→1
- **上浮**: translateY(8px)→0
- **缩放**: scale(0.98)→1
- **颜色**: 过渡而非突变

### 3. 禁止
- 剧烈闪烁
- 过度弹跳
- 花哨的3D效果

---

## 七、响应式断点

```css
--mobile: 480px;
--tablet: 768px;
--desktop: 1024px;
--wide: 1440px;
```

### 布局调整
- **画廊**: PC 端多列，移动端减少列数
- **侧边栏**: PC 显示，移动端隐藏或底部
- **字体**: 等比缩放

---

## 八、无障碍 (A11y)

- 对比度: 文字与背景满足 WCAG AA
- 键盘导航: 支持 Tab 切换
- 焦点指示: 可见的 focus 样式
- 图片 alt: 描述性文本
