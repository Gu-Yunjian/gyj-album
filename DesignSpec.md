> 版本: 1.0

## 一、设计哲学

### 1.1 核心原则

| 原则 | 描述 | 应用示例 |
|------|------|----------|
| **极简主义** | 大留白，减少视觉噪音，让照片成为主角 | 导航栏仅保留必要文字链接 |
| **沉浸体验** | 平滑的交互，不打断浏览节奏 | 灯箱使用淡入淡出过渡 |
| **内容优先** | 弱化 UI 控件，强化内容展示 | 照片信息使用小字号、低对比度 |
| **一致性** | 统一的视觉语言和交互模式 | 所有页面使用相同的间距系统 |

### 1.2 设计目标
- 打造一个**专业、克制、有质感**的摄影作品集
- 让访客专注于摄影作品本身
- 提供流畅、自然的浏览体验

## 二、色彩系统

### 2.1 亮色模式 (Light Mode)

```css
--bg-primary: #FFFFFF;      /* 主背景 - 纯白 */
--bg-secondary: #FFFFFF;    /* 次级背景 */
--bg-card: #FFFFFF;         /* 卡片背景 */

--text-primary: #000000;    /* 主文字 */
--text-secondary: rgba(0, 0, 0, 0.5);  /* 次级文字 */
--text-muted: rgba(0, 0, 0, 0.5);      /* 弱化文字 */

--border: rgba(0, 0, 0, 0.1);
--shadow: rgba(0, 0, 0, 0.06);
--accent: #000000;
--accent-hover: rgba(0, 0, 0, 0.7);
```

### 2.2 暗色/夜间模式 (Dark Mode)

```css
/* 影集内容页夜间模式变量 */
--bg-primary: #1a1a1a;      /* 主背景 */
--bg-secondary: #242424;    /* 次级背景 */
--bg-card: #2a2a2a;         /* 卡片背景 */
--text-primary: #e5e5e5;    /* 主文字 */
--text-secondary: #999999;  /* 次级文字 */
--text-muted: #666666;      /* 弱化文字 */
--border: #3a3a3a;
--shadow: rgba(0, 0, 0, 0.5);
```

**注意**: 影集内容页夜间模式下，左侧主展示区背景应为纯黑色 `#000000`

### 2.3 使用规范

| 元素 | 颜色变量 | 说明 |
|------|----------|------|
| 页面背景 | `--bg-primary` | 纯白或近黑 |
| 正文文字 | `--text-primary` | 标题、重要内容 |
| 辅助文字 | `--text-secondary` | 描述、副标题 |
| EXIF 信息 | `--text-muted` | 技术参数，最小化视觉权重 |
| 边框 | `--border` | 极淡的分隔线 |
| 交互 Hover | `--accent-hover` | 文字链接变色 |

### 3.3 导航栏特殊规范

## 五、组件设计规范

### 5.1 导航栏 (Navigation)

**结构**:
```
┌─────────────────────────────────────────┐
│ GU YUN-JIAN PROJECTS    影集    关于    │
└─────────────────────────────────────────┘
```

**样式**:
- 高度: 60px
- 位置: fixed, top: 0
- 背景: `rgba(255, 255, 255, 0.8)` + `backdrop-filter: blur(8px)`
- 底部边框: `box-shadow: 0 1px 0 rgba(0, 0, 0, 0.05)`
- Z-index: 1000

**交互**:
- 链接 Hover: 颜色从 `rgba(0,0,0,0.5)` 变为 `#000000`
- 下划线动画: 宽度从 0 到 100%，时长 200ms

### 5.2 照片卡片 (PhotoCard)

**首页布局**:
- 瀑布流/网格布局
- 无圆角（保持照片原貌）
- 无阴影（保持简洁）
- Hover: 轻微放大 `scale(1.02)` + 阴影加深

**尺寸**:
- 移动端: 180px 宽度
- PC 端: 240px 宽度
- 高度: 自适应（保持比例）

### 5.3 影集内容页布局

**双栏布局**:
```
┌────────────────────┬─────────────────┐
│                    │  影集标题       │
│                    │  副标题         │
│                    │                 │
│    照片展示区       │  ─────────────  │
│    (75% 宽度)       │  照片标题       │
│                    │  照片描述       │
│                    │  EXIF 信息      │
│                    │                 │
│                    │  [缩略图导航]   │
│                    │  1 / 5          │
└────────────────────┴─────────────────┘
```

**左侧照片区**:
- 背景: 亮色 `#FFFFFF`，夜间 `#000000`
- 照片居中，最大宽度 1200px
- 导航箭头: 圆形按钮，左右两侧

**右侧信息区**:
- 宽度: 320px (PC)
- 背景: `--bg-primary`
- 缩略图: 60x60px，间隙 8px

### 5.4 灯箱 (Lightbox)

**首页灯箱**:
- 全屏黑色遮罩 `rgba(0, 0, 0, 0.92)`
- 照片居中，最大 90vw x 85vh
- 底部信息浮层: 渐变遮罩 + 文字
- 显示: 照片标题 + EXIF (光圈 · 快门 · ISO)
- **不显示**: 计数器 (如 "16/30")

**关闭按钮**: 右上角，32x32px，白色半透明
**导航箭头**: 左右两侧，40x40px

### 5.5 EXIF 信息展示

**格式**: `f/8 · 1/60s · ISO 100`

**样式**:
- 字号: 13px
- 颜色: `--text-muted`
- **无背景色块**（已移除）
- 仅显示光圈、快门速度、ISO 三项

## 六、动画与过渡

### 6.1 过渡时长

```css
--transition-fast: 150ms ease-out;    /* 微交互 */
--transition-base: 200ms ease-out;    /* 常规 */
--transition-slow: 300ms ease-in-out; /* 页面级 */
```

### 6.2 动画类型

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面淡入 | opacity: 0→1 | 400ms | ease-in-out |
| 元素上浮 | translateY(30px)→0 | 400ms | ease-out |
| 缩放进入 | scale(0.95)→1 | 300ms | ease-out |
| 颜色过渡 | - | 200ms | ease |
| 导航下划线 | width: 0→100% | 200ms | ease-out |
| 照片 Hover | scale(1)→(1.02) | 300ms | cubic-bezier |

### 6.3 性能优化

- 使用 `transform` 和 `opacity` 实现动画（GPU 加速）
- 避免动画 `width`, `height`, `top`, `left`
- 适当使用 `will-change`

### 7.1 断点定义

```css
--mobile: 768px;
--tablet: 1024px;
--desktop: 1440px;
```

### 7.2 布局适配

| 页面 | PC | 平板 | 移动端 |
|------|-----|------|--------|
| 首页 | 多列瀑布流 | 2-3 列 | 2 列 |
| 影集列表 | 网格 | 网格 | 单列 |
| 影集内容 | 双栏 | 单栏 | 单栏 |

### 7.3 移动端特殊处理

- 导航箭头隐藏（使用滑动手势）
- 侧边栏移至底部或隐藏
- 字号等比缩小

## 八、无障碍 (A11y)

### 8.1 色彩对比度
- 主文字与背景对比度 >= 4.5:1 (WCAG AA)
- 辅助文字与背景对比度 >= 4.5:1

### 8.2 交互无障碍
- 所有交互元素支持键盘导航
- Focus 状态可见（outline: 2px solid）
- 图片必须有 alt 描述

### 8.3 图片保护
- 禁止右键菜单
- 禁止拖拽保存
- 禁止文本选择

- [ ] 颜色使用符合色彩系统规范

- [ ] 间距使用变量系统中的值
- [ ] 动画时长和缓动符合规范
- [ ] 响应式适配各断点
- [ ] 暗色模式样式已定义（如适用）
- [ ] 交互状态（hover, focus）已定义
