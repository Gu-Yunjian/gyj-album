# GU-Album 产品结构说明文档

> 版本: 1.0  
> 最后更新: 2026-03-11  
> 产品定位: 个人摄影作品集展示网站

---

## 一、产品概述

### 1.1 产品定位
GU-Album 是一个**极简风格的个人摄影作品集网站**，旨在以沉浸式的浏览体验展示摄影作品，让访客专注于照片本身。

### 1.2 核心功能

| 功能模块 | 描述 | 优先级 |
|----------|------|--------|
| 首页画廊 | 瀑布流展示精选照片 | P0 |
| 影集管理 | 按主题/时间组织照片 | P0 |
| 灯箱浏览 | 全屏查看照片详情 | P0 |
| EXIF 展示 | 显示拍摄参数 | P1 |
| 夜间模式 | 影集页暗色主题 | P1 |
| BGM 播放 | 影集背景音乐 | P2 |

### 1.3 目标用户
- 摄影爱好者浏览作品
- 潜在客户查看样片
- 个人品牌展示

## 三、页面结构详解

### 3.1 首页 (`/`)

**页面类型**: 客户端渲染 (CSR)

**布局**:
```
┌──────────────────────────────────────┐
│ 导航栏 (固定顶部)                      │
├──────────────────────────────────────┤
│                                      │
│      可拖动画布                        │
│   ┌────┐ ┌────┐ ┌────┐              │
│   │照片│ │照片│ │照片│   ...         │
│   └────┘ └────┘ └────┘              │
│                                      │
│   (个人简介卡片 - 画布中心)            │
│                                      │
├──────────────────────────────────────┤
│ 页脚版权信息                           │
└──────────────────────────────────────┘
```

**功能**:
- **无限画布**: 可拖动浏览，惯性滑动
- **瀑布流**: 照片按原始比例排列
- **灯箱**: 点击照片全屏查看
- **个人简介**: 固定于画布中心

**交互**:
- 鼠标拖动/触摸滑动移动画布
- 点击照片打开灯箱
- 灯箱内左右切换照片

**数据来源**:
- 所有影集照片 + `/public/home-photos/` 照片

---

### 3.2 影集列表页 (`/collections`)

**页面类型**: 服务端渲染 (SSR)

**布局**:
```
┌──────────────────────────────────────┐
│ 导航栏                                │
├──────────────────────────────────────┤
│                                      │
│   ┌─────────┐ ┌─────────┐           │
│   │ 影集封面 │ │ 影集封面 │   ...    │
│   │ 标题    │ │ 标题    │           │
│   └─────────┘ └─────────┘           │
│                                      │
└──────────────────────────────────────┘
```

**功能**:
- 网格展示所有影集
- 影集封面自动识别（优先 cover.jpg，否则第一张照片）
- Hover 显示影集标题

**数据结构**:
```typescript
{
  name: "beijing-hutong",
  title: "北京胡同",
  subtitle: "2024年秋日随拍",
  cover: "cover.jpg"
}
```

---

### 3.3 影集内容页 (`/album/[name]`)

**页面类型**: 服务端渲染 (SSR) + 客户端交互

**布局**:
```
┌────────────────────────┬─────────────┐
│  [X] [☀/🌙]            │             │
│                        │  影集标题    │
│                        │  副标题      │
│                        │             │
│                        │  ─────────  │
│     照片展示区          │  照片标题    │
│     (点击无反应)        │  照片描述    │
│                        │  EXIF 信息   │
│   [←]            [→]   │             │
│                        │  [缩略图]   │
│                        │  1 / 5      │
└────────────────────────┴─────────────┘
```

**左侧 - 照片展示区**:
- 当前照片大图展示
- 导航箭头（左右切换）
- **点击照片无反应**（无灯箱）
- 夜间模式背景为纯黑

**右侧 - 信息面板**:
- 影集标题和副标题
- 当前照片标题和描述
- EXIF 信息（光圈 · 快门 · ISO）
- 缩略图导航
- 照片计数器
- BGM 控制（如有背景音乐）

**控制按钮** (左上角):
- 关闭按钮 (X): 返回影集列表
- 主题切换 (☀/🌙): 日间/夜间模式

**夜间模式样式**:
```css
左侧背景: #000000
右侧背景: #1a1a1a
文字颜色: #e5e5e5 (主), #999999 (次)
```

## 四、数据模型

### 4.1 照片存储结构

```
public/
├── photos/                      # 影集照片
│   └── [album-name]/            # 影集目录
│       ├── info.txt             # 影集元数据
│       ├── cover.jpg            # 封面 (可选)
│       ├── bgm.mp3              # 背景音乐 (可选)
│       └── *.jpg                # 照片文件
└── home-photos/                 # 首页精选
    └── *.jpg
```

### 4.2 影集元数据 (`info.txt`)

```json
{
  "title": "影集标题",
  "subtitle": "副标题描述",
  "photos": {
    "filename1": {
      "title": "照片1标题",
      "desc": "照片1描述"
    },
    "filename2": {
      "title": "照片2标题",
      "desc": "照片2描述"
    }
  }
}
```

### 4.3 EXIF 数据

**展示字段**:
- 光圈 (Aperture): `f/8`
- 快门速度 (Shutter Speed): `1/60s`
- ISO: `ISO 100`

**格式**: `f/8 · 1/60s · ISO 100`

## 五、用户流程

### 5.1 浏览流程

```
进入首页
    │
    ├──► 浏览瀑布流照片
    │       │
    │       ├──► 点击照片 ► 灯箱查看
    │       │       │
    │       │       ├──► 左右切换照片
    │       │       │
    │       │       └──► 关闭灯箱
    │       │
    │       └──► 拖动画布探索更多
    │
    ├──► 点击导航"影集"
    │       │
    │       └──► 查看影集列表
    │               │
    │               └──► 点击影集 ► 进入影集内容页
    │                       │
    │                       ├──► 左右切换照片
    │                       ├──► 查看 EXIF 信息
    │                       ├──► 切换夜间模式
    │                       └──► 点击缩略图跳转
    │
    └──► 点击导航"关于"
            └──► 查看个人介绍
```

### 5.2 交互矩阵

| 页面 | 点击照片 | 拖动 | 键盘 | 其他 |
|------|----------|------|------|------|
| 首页 | 打开灯箱 | 移动画布 | ESC 关闭灯箱 | - |
| 影集列表 | 进入影集 | - | - | - |
| 影集内容 | 无反应 | - | ← → 切换照片 | 点击缩略图切换 |

## 七、设计决策记录

### 7.1 首页 vs 影集页交互差异

**决策**: 首页有灯箱，影集页无灯箱

**理由**:
- 首页作为画廊入口，需要灯箱沉浸式浏览
- 影集页右侧已有详细信息面板，灯箱重复
- 影集页专注于单张照片的深度展示

### 7.2 EXIF 展示简化

**决策**: 仅展示光圈、快门、ISO

**理由**:
- 这三项是摄影技术参数的核心
- 其他信息（相机型号、镜头）对观者价值较低
- 保持界面简洁

### 7.3 夜间模式仅影集页

**决策**: 夜间模式仅在影集内容页可用

**理由**:
- 影集页浏览时间长，需要护眼模式
- 首页为浅色极简风格，保持统一
